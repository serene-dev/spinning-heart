
# Spinning, beating heart

Youtube link: https://youtu.be/wsf6ORBDcfg

Compilation and running:

C
```
$ gcc -o spinning_heart spinning_heart.c
$ ./spinning_heart
```
Rust
```
$ rustc spinning_heart.rs
$ ./spinning_heart
```
Java
```
$ javac SpinningHeart.java
$ java SpinningHeart
```
Python
```
$ python3 spinning_heart.py
```
